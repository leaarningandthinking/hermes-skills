# hermes-skills

[Hermes Agent](https://github.com/NousResearch/hermes-agent) 中文社区精选技能合集，由 [hermesai.top](https://www.hermesai.top) 维护。

## 安装

在终端运行：

```bash
hermes skills install https://raw.githubusercontent.com/leaarningandthinking/hermes-skills/main/<技能名>/SKILL.md
```

例如安装 `codebase-inspection`：

```bash
hermes skills install https://raw.githubusercontent.com/leaarningandthinking/hermes-skills/main/codebase-inspection/SKILL.md
```
