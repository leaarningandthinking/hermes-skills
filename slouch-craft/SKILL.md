# 摸鱼大师 Slouch-Craft 🎣

专业级职场摸鱼助手，让你在老板和同事眼中始终是那个"看起来很忙"的人。

## 功能

### 1. 伪装代码生成器 (fake-code)
生成看起来超级专业、实则毫无意义的代码。支持多种语言：
- Python（数据分析、机器学习风格）
- JavaScript（React/Vue 复杂组件风格）
- Java（企业级架构风格）
- SQL（复杂查询风格）

### 2. 伪终端伪装 (fake-terminal)
创建伪装成重要系统处理的可视化界面：
- 系统升级中...
- 正在下载大型依赖...
- 正在部署生产环境...
- 正在执行复杂算法...

### 3. 摸鱼雷达 (slouch-radar)
快速趣味内容生成：
- 随机颜文字 (つ✧ω✧)つ
- 趣味段子/梗
- 今日运势
- 解压小游戏

### 4. 紧急撤退 (panic-mode)
一键切换到"工作模式"：
- 生成紧急任务提醒
- 创建看起来重要的文档标题
- 生成"看起来很忙"的邮件/消息模板

### 5. 下班倒计时 (clock-watch)
显示距离下班时间的倒计时，支持自定义工作时间

## 使用方法

使用命令 `slouch` 作为入口，或者直接描述你的需求。

### 1. 伪装代码 (fake-code)
- `python scripts/fake_code.py python` - Python 代码
- `python scripts/fake_code.py js` - JavaScript 代码
- 支持：Python、JavaScript

### 2. 伪终端 (terminal)
- `python scripts/fake_terminal.py upgrade` - 系统升级
- `python scripts/fake_terminal.py deploy` - 部署生产环境
- `python scripts/fake_terminal.py analytics` - 数据分析
- `python scripts/fake_terminal.py api` - API 测试

### 3. 摸鱼内容 (funny)
- `python scripts/funny_generator.py` - 显示颜文字、运势、段子、推荐网站

### 4. 紧急撤退 (panic)
- `python scripts/panic_mode.py` - 一键生成工作模式界面

### 5. 下班倒计时 (countdown)
- `python scripts/countdown.py` - 默认18:00下班
- `python scripts/countdown.py 17` - 17:00下班
- `python scripts/countdown.py 17 30` - 17:30下班

## 脚本

- `scripts/fake_code.py` - 伪装代码生成器
- `scripts/fake_terminal.py` - 伪终端界面
- `scripts/funny_generator.py` - 趣味内容生成器
- `scripts/panic_mode.py` - 紧急撤退生成器
- `scripts/countdown.py` - 下班倒计时
